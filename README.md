# babel-voip-application

Description
--

VOIP application with a server/client architecture, similar to Skype made in C++.

Dependencies
--
Conan
Portaudio
OpusCodec
Boost
Qt5.9 or above

Usage
--
babel_server and babel_client built using cmake
